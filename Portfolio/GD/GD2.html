Literal Typography

Overview

For this project, I had to choose three words to create "literal typography". The word was written in a appropiate font, and then edited and added to until it clearly displays the definition. For example, the word "Melt" is melting like a candle that is burning down. Things to consider for this project was the chosen font, capitalization, how well the definition is conveyed, and general aesthetics.

Tools and Technologies

This project was created using Adobe Illustrator.
