Logo Design

Overview

The goal of this project was to take a photograph and then simply it down to a simple, black logo design for a company. I started with the idea of an athletic swimsuit company, and a photograph of a jellyfish. Then, I drew the jellyfish in Illustrator following the photograph but simplifying it slightly. Next, it needed to be simplified more so it was less a jellyfish and more a symbol that resembled a jellyfish. Also, I began applying the faux company's style to it, such as curling the tentacles more and added a wave pattern into the body portion. Finally, different font combinations were tested until I settled on one for the final logo.

Tools and Technologies

This project was created using Adobe Illustrator, with initial sketches done traditionally.

Challenges

The largest challenge of this project was dealing with the company and "client" I had come up with. I had to constantly think of and apply the feeling I wanted to portray of a modern, dynamic company.
