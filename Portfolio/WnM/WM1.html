Overview

This site was created around a research paper I wrote about the current state of wearable technology, in which I argued wearable technology has not quite reached it's full potential, while also arguing about privacy issues and other concerns. The site contains an introduction page, three pages containing the main arguements, and a sources page for all research used in the paper and on the site.

Tools and Technologies

This project uses HTML5 and CSS, and was uploaded to the banjo.rit.edu server using FileZilla

Challenges

This site was made with only a week's experience using HTML and CSS
